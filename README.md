# Install

1) Fork the repository;
2) Go "Actions";
3) Click the green button;
4) Click "Build WSA";
5) Click "Run worflow";
6) select in "Build Arch" your bit system;
7) select in "Variants of gapps" - "pico";
8) select in "Root solution" - "none";
9) Done.
